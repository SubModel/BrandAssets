## Asset Categories

### Icon Versions (256×256)
- **Colored Icon**: Use for general purposes, web applications, and documents
- **Monochrome Icon**: Use for single-color applications, watermarks, or when contrast is needed

### Horizontal Logo Versions
- **Large (1024×172)**: Use for headers, banners, and high-resolution applications
- **Small (256×43)**: Use for compact spaces, navigation bars, or thumbnails
- **Monochrome**: Use for single-color printing or dark backgrounds
- **White**: Use specifically for dark backgrounds

## Usage Guidelines

### Recommended Use Cases

| Asset Type | Recommended Usage |
|------------|------------------|
| Colored Icons | Web favicons, app icons, documentation |
| Monochrome Icons | Watermarks, single-color prints, embossing |
| Large Horizontal Logos | Website headers, presentation slides, banners |
| Small Horizontal Logos | Navigation bars, compact layouts, thumbnails |
| White Logos | Dark backgrounds, overlays, video content |

### File Format Guidelines

- **PNG**: Use for web applications, documents, and when transparency is needed
- **SVG**: Use for scalable applications, high-DPI displays, and print materials
